    # deskmini 介绍

    Pasted image 20230416112309.png

    1. 小型化APU平台，体积仅1.92升
    2. 最大支持内存64G
    3. 2x M.2 PCIe Gen3 x4 + 2x 2.5" SATA 6Gb SSD/HDD
    4. 可以将其中一个M.2接口转接有线网卡

    # 具体配置

    1. CPU: AMD Ryzen™ 5 3400G with Radeon™ RX Vega 11 Graphics（938 ¥）
    2. MEM: DDR4 64G 3200MHz （800 ¥）
    3. 准系统+机箱：DeskMini A300 （987 ¥）
    4. 硬盘: 256G（NVME） + 4T（2x SATA 2.5" SSD）（265 ¥ + 1800 ¥）
    5. 风扇: 猫头鹰 NF-A9x14 PWM 9cm （125 ¥）

    总价：4915 ¥

    纯SSD硬盘+猫扇：静音，几乎没有声音，可直接放在卧室。

    # 操作系统

    # Gentoo Linux

    优点：

    1. 从源码构建
    2. 稳定性高
    3. 自由度高
    4. 滚动升级
    5. 软件包多且新

    缺点：

    1. 从源码构建费时、费电
    2. 安装、配置复杂

    # openwrt

    无须自己编译，直接使用别人构建好的镜像与软件包，安装软件跟其他linux系统一样简单

    1. 代理：openclash
    2. 加速器：uu游戏加速器
    3. VPN组网：zerotier/tailsacle

    # 软件

    1. 文件系统：ext4(btrfs) + zfs
    2. 网络文件共享：NFS + samba
    3. 系统管理：smartd, nginx等
    4. 虚拟机：libvirt + qemu + kvm
    5. 下载：aria2 + qbittorrent

    # 容器

    docker + k8s

    1. 基础设施：cert-manager, ingress, metallb, prometheus-stack, EFK
    2. 媒体服务器：plex/emby/jellyfin
    3. RSS： Awesome TTRSS + RSSHUB
    4. 管理面板：portainer + kubesphere
    5. 数据库：PostgreSQL, Redis, Elasticsearch, couchdb
    6. 其他应用：vaultwarden/bitwarden, umami等

    # 缺失

    1. 图片服务器
    2. 漫画服务器
