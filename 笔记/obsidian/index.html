  • 三大原则
    • 链接
      • link first, folder least.
      • 尽量少的建立层级
      • 通过“中枢”方式创建新笔记
    • 标签
      • 标题分类，树状、namespace 类似
      • 标签位置，font matter 或最前、最后
      • 标签要分的很碎，只讲一个东西。精细的组合＞全面的概括。层级无限扩展，适当就好
      • 模板
        • 思维导图：在原有文件中写清楚结构，然后导入为 markdown
        • 插件：advanced cursors、templater、quickadd
        • quickadd：capture 快速添加
        • 慎重定结构灵活用片段

# 快捷键

  • Cmd-E ：编辑、预览模式切换
  • 按住 Cmd 点击链接：在新面板中打开该链接
  • 编辑模式下按住 Cmd 再单击链接：跳转链接
  • Cmd-alt-左方向键 或 鼠标侧键：回到之前的页面
  • Cmd-P：打开命令面板
  • Ctrl-Enter：切换 list 和 todo
  • Ctrl-T：编辑或新建 task
  • Ctrl-D：打开或新建当天日记
  • Alt+鼠标单击：多光标操作
