    # 双向连接

    B 站最强 Notion 测评，极致丝滑！新手进阶必看 16 个教程用法丨笔记软件推荐_哔哩哔哩_bilibili

    # Notion

    B 站最强 Notion 测评，极致丝滑！新手进阶必看 16 个教程用法丨笔记软件推荐

    # 优点

    1. 数据库、视图功能强大。
    2. 排版美观，
    3. 使用便捷。

    # 缺点

    1. 格式不通用，虽然能导出markdown，但会损失不少信息。
    2. 双链较弱
    3. 没有大纲。
    4. 有时会卡、慢。
    5. 不支持嵌入B站视频
    6. 支持的标题层级太少

    # 适合

    1. notion适合系统的、线性的、有逻辑顺序的创作，例如日记、工作文档、学习笔记、搭建知识系统。

      notion也适合用来组织、整理和分享收集和创建的素材和图文作品。但notion不适合碎片化的记录方式。

    # Obisidian

    先记下来，插件、方法论可以后面再配置、整理。

    发布

    下次详细介绍。

    也许是 B 站最好的 Obsidian 新手教程！爆肝30天，一站式入门双向链接笔记软件

    # 优点

    1. 双链汇总关键信息。
    2. 传统的文件夹结构
    3. 插件多

    # 缺点

    1. 基本只支持纯markdown，原生不支持分栏，插件实现分栏太麻烦。排便不方便、不美观。双链预览窗口不如思源大。不支持数据库-视图？
    2. 无法嵌入播放视频，只能在阅读模式中打开，并且插件还有bug，切换阅读和编辑模式会不断弹出新窗口
    3. PDF读书笔记几乎无法使用，markMind与annotator均会在编辑模式下显示一坨乱七八糟的东西，并且annotator打开大PDF时非常卡，并且不方便实现跳转。
    4. 视频笔记功能的插件有一些bug。

    # 用法

    1. 每天日记中的待办事项集中管理。
    2. 读书笔记，深度阅读，PDF批注，串联笔记、用双链拓展读书笔记
    3. 阅读清单整理。
    4. 视频笔记，支持youtube、bilibli，支持跳转时间戳
    5. 记录日报周报
    6. 编年史记

    # 插件

    1. dataview：与notion的数据库-视图功能类似，但感觉还是比不过，但胜在类SQL语句灵活。
    2. QucikAdd、Templater：模版、快速添加
    3. Charts：生成各种图表

    2. 开源

    # 缺点

    1. 不好看。
    2. 不成熟。

    # Logseq

    爆肝100小时，全网最易懂的 Logseq 指南！双向链接笔记软件科普

    block 为最小单位。

    # 优点

    1. 超级日记本：支持嵌入代码、表格、视频、画板
    2. 可以嵌入视频做视频笔记，youtube、bilbili支持时间戳跳转，感觉比obsidian做的好，嵌入的bilbili视频呢不支持1080
    3. PDF读书笔记做的比obsidian好
    4. 有插件系统
      1. 支持flomo的热力图
      2. 支持将大纲转换成思维导图，可以放弃幕布
    5. 开源
    6. 有潜力

    # 缺点

    1. 插件太少
    2. 强制大纲模式，所有行开头都有 -
    3. 长文体验不好
    4. 没有文件夹结构
    5. 卡
    6. 多平台同步没有免费方案
    7. 原生不支持在侧边栏显示大纲（跟 notion 一样）

    # 用法

    1. 日志
    2. 任务管理系统
    3. 工作日报周报
    4. 读书笔记，比obsidian做的好

    # 特点

    1. 积少成多，零散的将内容记录在每天日记中，再通过双链将每天日记中值得关注点汇总
