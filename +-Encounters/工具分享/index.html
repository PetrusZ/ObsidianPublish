    # Deskmini 介绍

    1. 小型化APU平台，体积仅1.92升
    2. 最大支持内存64G
    3. 2x M.2 PCIe Gen3 x4 + 2x 2.5" SATA 6Gb SSD/HDD
    4. 可以将其中一个M.2接口转接有线网卡

    # 具体配置

    1. CPU: AMD Ryzen™ 5 3400G with Radeon™ RX Vega 11 Graphics（938 ¥）
    2. MEM: DDR4 64G 3200MHz （800 ¥）
    3. 准系统+机箱：DeskMini A300 （987 ¥）
    4. 硬盘: 256G（NVME） + 4T（2x SATA 2.5" SSD）（265 ¥ + 1800 ¥）
    5. 风扇: 猫头鹰 NF-A9x14 PWM 9cm （125 ¥）

    总价：4915 ¥

    纯 SSD 硬盘+猫扇：静音，几乎没有声音，可直接放在卧室；性能强劲，8C16T CPU，64G 内存。

    # 软件

    1. 文件系统：ext4(btrfs) + zfs
      1. ext4 稳定，传统 linux 文件系统格式
      2. zfs 支持 raid、snapshot
    2. 网络文件共享：NFS + samba
    3. 系统管理：smartd, nginx等
      1. smartd 硬盘状态检测
      2. nginx 反代
      3. acme.sh 自签证书
    4. 虚拟机：libvirt + qemu + kvm
      1. 传统 linux 虚拟机
      2. libvirt xxx
      3. qemu xxx
      4. kvm xxx
    5. 下载：aria2 + qbittorrent
      1. aria2 http 下载
      2. qbittorrent bt/pt 下载

    # 容器

    docker + k8s

    1. 基础设施：cert-manager, ingress, metallb
    2. 监控：netdata, prometheus-stack, EFK
    3. 媒体服务器：plex / emby / jellyfin
    4. RSS： Awesome TTRSS + RSSHUB
    5. 管理面板：portainer + kubesphere
    6. 数据库：PostgreSQL, Redis, Elasticsearch, Mongodb, Couchdb
    7. 其他应用：vaultwarden/bitwarden, umami等
